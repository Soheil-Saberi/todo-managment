# Eco Task

##### todo managment

## Installation & Run Locally

Clone the project

```bash
  git clone https://github.com/Soheil-Saberi/todo-managment.git
```

Go to the project directory

```bash
  cd todo-managment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open http://localhost:3000
